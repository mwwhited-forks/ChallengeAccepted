# Card Shuffle

## Summary

Many games are based around a deck of cards.  A common routine is shuffling 
that deck into a random order before dealing/playing hands.  

## Challenge

* Write a method can take a ordered deck of cards and shuffle them into a random order.
* The number of cards and sets are up to you but a common deck would be 4 suits 
  (Hearts, Diamonds, Spades and Clubs) with 13 cards 2-10, Jack, Queen, King, Ace

## Bonus

After shuffling the deck deal out the cards into multiple hands like you would for 
Blackjack or another poker game.
